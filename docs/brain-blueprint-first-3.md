# Brain Blueprint: First 3 Brains

## 1. Pitch Brain

### Raw substrate
- chromatic pitch universe
- octave space
- enharmonic spellings
- interval distance

### Primitive distinctions
- same / different
- up / down
- near / far
- stable / unstable
- inside / outside collection

### Organization layer
- pitch classes
- scales
- modes
- collections
- register zones
- instrument range filters

### Functional layer
- anchor tones
- passing tones
- chord tones
- tendency tones
- contour targets
- register arcs

### Intent layer
- education: scale drills, interval practice, range-limited reading
- composition: melody search, contour writing, color-tone exploration
- randomization: controlled pitch variety
- evaluation: best sounding contour or register choice

### Inputs
- key / mode
- harmony targets
- phrase role
- placement role
- user prompt

### Outputs
- pitch candidates
- pitch ranking
- selected pitch events

### Future expansions
- modulation
- chromatic voice-leading
- instrument-specific playability
- style-aware pitch behavior

---

## 2. Rhythm Brain

### Raw substrate
- pulse grid
- beat positions
- subdivisions
- duration units

### Primitive distinctions
- long / short
- attack / sustain
- even / uneven
- simple / compound

### Organization layer
- duple
- triple
- compound
- dotted systems
- tuplets
- meter families
- subdivision families

### Functional layer
- density
- continuity
- cadence duration shaping
- phrase support
- groove character
- readability level

### Intent layer
- education: beginner reading, subdivision practice, syncopation drills
- composition: groove search, rhythmic color, phrase pacing
- randomization: idea generation across families
- evaluation: most readable vs most compelling rhythm

### Inputs
- meter
- placement logic
- phrase role
- prompt vocabulary
- difficulty / intent

### Outputs
- duration vocabulary
- measure-fit rhythm choices
- rhythm families and density profile

### Future expansions
- nested tuplets
- composite meter
- polymeter / polyrhythm
- style-specific groove templates

---

## 3. Placement Brain

### Raw substrate
- event location on the time grid
- downbeats and offbeats
- sustained-over-beat events

### Primitive distinctions
- onBeat / syncopated
- arrival / continuation
- grounded / displaced

### Organization layer
- beat anchors
- syncopation families
- anticipation points
- suspension-like holds

### Functional layer
- anchor notes on structural moments
- passing or color notes in displaced moments
- cadence arrival placement
- opening stability
- middle tension / release shaping

### Intent layer
- education: teach on-beat vs offbeat reading
- composition: control tension and groove placement
- randomization: vary placement balance
- evaluation: compare grounded vs displaced phrase versions

### Inputs
- rhythm events
- harmony targets
- phrase sections
- prompt intent

### Outputs
- placement tags per event
- pitch-behavior guidance by placement
- rhythmic tension profile

### Future expansions
- strong / medium / weak beat hierarchy under onBeat
- anticipation classifier
- suspension / delay behavior
- style-aware placement maps

---

## Interaction summary
- Pitch Brain decides what notes are available and which ones are likely meaningful.
- Rhythm Brain decides how time is divided and grouped.
- Placement Brain decides where events feel anchored or displaced within that time.

Together they create the first true musical language scaffold:
- Pitch = alphabet
- Rhythm = time syntax
- Placement = stress / speech pattern
