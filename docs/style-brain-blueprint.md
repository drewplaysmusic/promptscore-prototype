# Style Brain Blueprint

## Core purpose
Style Brain maps the full universe of music across time periods, genres, traditions, and composer behaviors into a structured system that can guide the other brains.

It should not simply copy a composer. It should organize stylistic behaviors into reusable constraints and tendencies.

---

## 1. Raw substrate
The raw substrate is the broad musical universe:
- time periods
- cultural traditions
- genres
- forms
- instrumentation conventions
- composer-specific tendencies

This is the stylistic equivalent of the chromatic pitch universe in Pitch Brain.

---

## 2. Primitive distinctions
Start with simple oppositions:
- simple / complex
- dense / sparse
- strict / flexible
- tonal / modal / post-tonal
- regular / irregular
- phrase-balanced / phrase-fluid
- grounded / expressive

These are the low-level traits underneath style.

---

## 3. Organizational layer
Style Brain should organize music by overlapping layers, not one flat label.

### Era layer
- Medieval
- Renaissance
- Baroque
- Classical
- Romantic
- Modern
- Contemporary

### Genre layer
- chorale
- sonata
- fugue
- folk song
- hymn
- jazz standard
- pop song
- film cue
- percussion etude
- groove study

### Behavior layer
- cadence style
- harmonic rhythm
- phrase symmetry
- motif development type
- rhythmic profile
- texture tendencies
- ornamental density

### Composer / substyle layer
- Bach chorale behavior
- Mozart phrase behavior
- Chopin lyric contour behavior
- blues shuffle behavior
- bebop line behavior

Style should be represented as a stack of tags and tendencies, not one string.

---

## 4. Functional layer
Style Brain should influence:
- Pitch Brain: interval and contour preferences
- Rhythm Brain: duration vocabulary and subdivision behavior
- Placement Brain: grounded vs syncopated tendencies
- Phrase Brain: phrase symmetry and cadence expectations
- Harmony Brain: progression behavior and cadence type
- Motif Brain: repetition and variation style
- Evaluation Brain: scoring weights by style goal

Style Brain is the global bias system that tells the other brains how a piece should behave.

---

## 5. Intent layer
### Education mode
- simplify style to teach a recognizable trait
- isolate one stylistic feature at a time
- compare eras or genres clearly

### Composer mode
- use style as inspiration, not imitation
- combine style tags intentionally
- compare alternate stylistic realizations

### Exploration mode
- hybridize styles
- test unusual cross-period pairings
- discover style blends

### Retrieval mode
- fetch public-domain references that match the requested style stack

---

## 6. Inputs
Style Brain should read:
- user prompt
- corpus metadata
- extracted musical features
- retrieved references
- target instrumentation if relevant

Example inputs:
- "Bach-like"
- "late Classical"
- "minor Baroque sequence"
- "romantic lyric phrase"
- "beginner hymn style"

---

## 7. Outputs
Style Brain should output:
- style tag stack
- behavior constraints
- weighting adjustments for other brains
- retrieval filters for corpus search

Example outputs:
- era = Baroque
- genre = chorale
- phrase symmetry = moderate
- motif repetition = high
- rhythmic freedom = low
- harmonic function priority = high

---

## 8. Suggested first version
### Version 1
Build a small style grouping system with:
- era
- genre
- composer reference tag
- a handful of behavioral tendencies

### Version 2
Allow hybrid style prompts:
- Baroque harmony + modern rhythm
- Classical phrase + folk simplicity

### Version 3
Use extracted corpus features to learn style tendencies automatically instead of manually assigning most behavior rules.

---

## 9. Relationship to corpus and retrieval
The corpus should store:
- era
- genre
- composer
- extracted behavior features

Style Brain should use that data to retrieve exemplars and translate the prompt into a style constraint stack.

This means Style Brain is the bridge between:
- the corpus
- the retrieval layer
- the musical brains

---

## 10. Style wheel concept
Pitch Brain has:
- chromatic universe -> scales/modes -> tonal function

Rhythm Brain has:
- pulse grid -> subdivisions -> rhythmic function

Style Brain should have:
- global music universe -> era/genre/composer layers -> behavioral function

This is the stylistic wheel or stylistic scaffold for the whole system.

---

## 11. Important principle
Do not reduce style to surface imitation.
Represent style as:
- behavior tendencies
- structural expectations
- phrase and cadence habits
- harmonic and rhythmic norms

That allows PromptScore to use references intelligently while staying creative and controllable.
