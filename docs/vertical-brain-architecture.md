# PromptScore Vertical Brain Architecture

## Core idea
Each brain should be built like a scaffolded vertical system:

- **substrate layer**: the raw musical universe under the brain
- **organization layer**: theory or grouping rules applied over the substrate
- **functional layer**: musical meaning and task behavior
- **user-intent layer**: education, composition, randomization, testing, generation

This mirrors the idea of a chromatic pitch wheel underneath, then scales/modes/chords placed on top.

## Elements of music mapped vertically

- **Pitch** -> chromatic substrate -> scales/modes/chords -> function -> intention
- **Rhythm** -> pulse substrate -> meter/subdivision systems -> placement/accent -> intention
- **Harmony** -> interval substrate -> chord systems -> tension/release -> intention
- **Melody** -> pitch + rhythm interaction -> contour/motif -> phrase function -> intention
- **Form** -> phrase grouping -> section logic -> development/cadence -> intention
- **Timbre/orchestration** -> sound source space -> instrument family organization -> role -> intention
- **Texture** -> event layering -> monophony/homophony/polyphony -> clarity/density -> intention
- **Dynamics/articulation** -> amplitude/attack space -> markings/groups -> expressive function -> intention

## Important principle
Every brain should answer four questions:
1. what is the raw musical space underneath?
2. how is that space organized?
3. what musical function does it serve?
4. how does the user's intention shape it?
